Install MV Divest

A Chrome extension — takes about 60 seconds to set up.

Step-by-step guide

1

Download & unzip the extension

Download the latest release ZIP from GitHub and unzip it anywhere on your computer — your Desktop or Downloads folder works fine.

📁  Downloads
📄 other-file.pdf
📂 MVdivest  — unzipped folder
📦 MVdivest.zip

The unzipped folder should contain manifest.json, dashboard.html, and other files.

2

Open Chrome Extensions

In Chrome, type chrome://extensions into the address bar and press Enter.

chrome://extensions
Extensions

You'll see a list of your installed extensions here.

3

Turn on Developer Mode

In the top-right corner of the Extensions page, toggle Developer mode on. This unlocks the ability to load extensions from your computer.

chrome://extensions
Extensions
Developer mode

Once enabled, three new buttons will appear at the top of the page.

4

Click "Load unpacked"

Click the Load unpacked button that appears after enabling Developer mode, then select the unzipped MVdivest folder.

chrome://extensions
📁  Select Extension Folder
📁 Desktop
📁 Documents
📂 MVdivest
5

MV Divest is installed

The extension card will appear on your Extensions page. It's ready to use.

chrome://extensions
💾
ManyVids Divest
Download your ManyVids catalogue with metadata export
6

Pin it to your toolbar (optional)

Click the puzzle-piece icon in the Chrome toolbar, find ManyVids Divest, and click the pin icon so it's always one click away.

💾
MV Divest is pinned to your toolbar
✓
You're all set. Click the MV Divest icon in your toolbar to open the dashboard. Make sure you're logged in to ManyVids in another tab before you start.
